# Agents folder

Drop a markdown (`.md`) file in here for each agent. The Agents page uses the
same layout as Skills — a browsable tree on the left, popular agents on the right.

Same frontmatter format. `category1` / `category2` nest the agent by what it's
for (e.g. `Software → Code Quality`, `Productivity → Research`).

```markdown
---
title: Code Reviewer
category1: Software
category2: Code Quality
description: Reviews diffs for bugs and suggests fixes.
---

# Code Reviewer

...content...
```

The manifest (`agents/index.json`) is regenerated by the `Build indexes` GitHub
Action on push, or locally with `node scripts/build-index.mjs`.
