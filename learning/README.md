# Learning folder

Drop a self-contained interactive **`.html`** learning doc in here. The Learning
page shows a quick intro card for each one; clicking a card opens the full doc in
a new tab.

Unlike the other sections (which use markdown), learning docs are standalone HTML.
Metadata is read from the `<head>`:

```html
<head>
  <title>Are You Smarter Than Claude?</title>
  <meta name="intro" content="A playful interactive deck — test your instincts against Claude." />
  <meta name="category1" content="AI Literacy" />
  <meta name="category2" content="Just for Fun" />
</head>
```

| Tag                          | Required | Purpose                          |
| ---------------------------- | -------- | -------------------------------- |
| `<title>`                    | yes      | Card title.                      |
| `meta[name=intro]`           | no       | Quick intro shown on the card.   |
| `meta[name=category1/2]`     | no       | Optional grouping labels.        |

Keep each doc fully self-contained (inline CSS/JS, or assets alongside it in this
folder) so it works when opened directly.

The manifest (`learning/index.json`) is regenerated by the `Build indexes` GitHub
Action on push, or locally with `node scripts/build-index.mjs`.
