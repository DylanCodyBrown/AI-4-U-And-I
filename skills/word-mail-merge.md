# Mail Merge in Word

Produce many personalized documents — letters, labels, emails — from a single
template and a data source such as an Excel sheet.

## When to use it

- Sending the same letter to many recipients with per-person details
- Printing name badges or address labels in bulk

## Steps

1. **Mailings → Start Mail Merge** and pick a document type.
2. **Select Recipients → Use an Existing List** and choose your spreadsheet.
3. Insert merge fields with **Insert Merge Field** where the data should appear.
4. **Preview Results** to check a few records.
5. **Finish & Merge** to print, email, or create a combined document.

## Tips

- Keep your data source's first row as clean column headers.
- Use **Rules → If…Then…Else** for conditional text.
