# GitHub MCP

A community server that exposes GitHub — repositories, issues, pull requests,
and Actions — as MCP tools your assistant can call.

## Why use it

- Triage issues and review PRs conversationally
- Pull file contents and CI status into context

## Setup

Point your client at the GitHub MCP server and provide a token with the scopes
you need. Prefer a fine-grained token limited to the repositories you want
exposed.
