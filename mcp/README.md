# MCP folder

Drop a markdown (`.md`) file in here for each MCP server. The MCP page builds its
left-hand tree from these files automatically.

Same format as `skills/`, but `category1` is the **source**:

- `Local` — servers that run on your own machine
- `Community` — public servers from the ecosystem
- `Homebrewed` — MCP servers you built yourself

```markdown
---
title: Filesystem
category1: Local
category2: Files
description: Read and write local files from your assistant.
---

# Filesystem MCP

...content...
```

The manifest (`mcp/index.json`) is regenerated by the `Build indexes` GitHub
Action on push, or locally with `node scripts/build-index.mjs`.
